Detailed Description

Auxiliary class used to throw exceptions.

It just defines <<= operator so that:

  • exceptions can be thrown calling std::throw_with_nested with a "throw like" syntax
  • precedence is the same as the throw operator (see https://en.cppreference.com/w/cpp/language/operator_precedence)
